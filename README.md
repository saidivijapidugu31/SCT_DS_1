# SCT_DS_1

## Task 1: Population Distribution Visualization

### Objective
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

### Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

### Project Description
This project visualizes population data using:
- Histogram for Age Distribution
- Bar Chart for Gender Distribution

### Outcome
The charts help understand the distribution of age groups and gender categories in the dataset.
